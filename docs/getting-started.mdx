# Getting Started

This guide will help you set up and run Surge rollup.

## Prerequisites

- Intel processor with SGX (Software Guard Extensions) support for the prover
- Docker and Docker Compose
- Git
- 16GB RAM minimum
- 50GB free disk space

## Steps

1. Install Kurtosis ([installation guide](https://docs.kurtosis.com/quickstart))
2. Deploy L1 ([guide](/docs/Guides/deploy-l1))
3. Deploy L2 ([guide](/docs/Guides/deploy-l2))
4. Set up Provers ([guide](/docs/Guides/provers/index))
5. Deploy Relayer ([guide](/docs/Guides/deploy-relay))
6. Launch Bridge UI ([guide](/docs/Guides/deploy-bridge-ui))
